3. The Concept — Module 1 Gate — Infection Control Integration

Module 1 Gate — Integrating Infection Control

This gate session combines Sessions 01–03: standard precautions, transmission-based isolation, and PPE sequence. NCLEX case studies do not announce the topic — you must recognize infection control issues inside broader patient scenarios and act in priority order.

Safety Checkpoint Flow (Every Patient Contact)

  1. Read the precaution sign / chart — know isolation type before entering
  2. Perform hand hygiene
  3. Don correct PPE for isolation type + procedure risk
  4. Deliver care using aseptic technique when indicated
  5. Doff PPE correctly — gloves first
  6. Hand hygiene again before leaving room or touching clean areas
  7. Dispose of equipment per policy; document as required

Gate Scenario — Walk Through

Scenario: You enter a room without reading the sign. The patient has draining MRSA wound. You touch the dressing with bare hands, then leave to see another patient.

What broke:

  • No isolation verification → contact precautions missed
  • No hand hygiene or PPE before contact
  • Cross-contamination risk to second patient

Correct priority action: Exit, verify precaution type, don gown and gloves, perform hand hygiene, re-enter using aseptic technique for wound care.

Sharps & Standard Precautions Reminder

  • Never recap needles — use one-handed scoop only if policy requires recapping
  • Found needle in linens → use forceps/tongs, place directly in sharps container
  • Standard precautions = all blood, all body fluids, mucous membranes, non-intact skin
  • Gloves are not a substitute for hand hygiene

Connection to NCLEX-RN Exam

Gate-style questions mirror NGN case studies: one scenario, multiple judgment calls. You will not be told "this is an infection control question." Recognize the safety issue, select the first action, and apply the least invasive correct intervention.
