Dependency Ordering

Each module builds on prior modules. Infection control and safety foundations precede all clinical specialties.

Infection Control → Emergency Response → Pharmacology → Prioritization
  → Cardiovascular → Respiratory → Endocrine → Pediatrics
    → Maternity → Mental Health → Capstone Review
