Rules We Follow

  1. Never jump ahead — clinical concepts taught in strict dependency order.
  2. One session = one focused topic = one commit.
  3. Always assess before intervening — nursing process first.
  4. Score below 80% on a quiz = repeat the session.
  5. Every study session ends with documentation updates.
  6. Understand every commit before proceeding.
  7. Practice questions test clinical reasoning, not memorization.
  8. Every lesson completable in 35–40 minutes.
